## [WIP] Quanto vale minha hora

A ideia deste projeto é facilitar na hora de fazer cálculo para saber o valor da sua hora trabalhada.
Espero que seja útil para você assim como é para mim. =)

### Desenvolvido com
  - HTML/CSS
  - ES6/Babel
  - Sass (.scss)
  - Webpack
  - Yarn

### Setup
```
$ yarn install
```

### Build
```
$ yarn dev
```
